# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/b9bdc4a5-67e1-4e6c-abc9-28ff868636d6

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a `.env` file by copying `.env.example` and set the `GEMINI_API_KEY` to your Gemini API key.
3. Run the app:
   `npm run dev`
